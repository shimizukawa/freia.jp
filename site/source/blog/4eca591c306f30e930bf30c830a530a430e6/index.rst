=============================
2005/12/24 今夜はラタトゥイユ
=============================

先週のクリスマス会で好評だったラタトゥイユをまた作りました。うまー。

作成時間15分。１鍋８００円。１皿で４００円くらい。ズッキーニだけで１本３００円します。次から茄子で代用してやる。
