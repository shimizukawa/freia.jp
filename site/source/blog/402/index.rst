=====================================
2007/02/18 モスのデミカツとチーフォン
=====================================

昼ごはんにモスで `デミカツドックとチーフォンドック`_ を食べる。どうせなので二つとも食べてみた。デミカツドックは揚げた手アツアツ。あふれんばかりのデミグラスソースがたまりません。ただしカツはメンチカツっぽい。チーフォンドックはバジルポテトソース(?)が微妙にミスマッチ。後に食べたからポテトのボリューム感が余計だったのかも・・・。


感想：１つで十分だった
