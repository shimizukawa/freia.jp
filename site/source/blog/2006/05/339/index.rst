=========================
東北の車窓から
=========================

秋田新幹線は在来線の線路を走ります。民家の横を走ったり、踏切があったり、新幹線すれ違い待ちがあったりします。こんなにいろいろあるのに、自由席はありません。


ところで、秋田ではちょうど桜が満開でした。さすがだ秋田。
