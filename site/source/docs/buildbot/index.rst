Buildbot
==========

buildbotはPythonベースの継続的インテグレーションシステムです。
サーバー・クライアントモデルで構成されており、リポジトリからのソース取得
からビルドとテストの実行までを様々なトリガーで実行します。
WebUIのみの提供でPythonが動作するプラットフォームであれば動作します。
