# イベント参加blog記事の執筆

参加メモからイベント参加blog記事を書き起こします。
ワークスペース内の過去記事（`site/source/blog/2024/`, `site/source/blog/2025/` 以下）を参照して、文体・構成を揃えてください。

## ファイル形式

reStructuredText (`.rst`)。先頭にメタデータを記載:

```rst
:date: YYYY-MM-DD HH:MM
:tags: PyCon, PyCon XX, 開催国名など
```

## 記事の構成（セクション順）

```
[タイトル]：「[イベント名] に参加しました」
[リード文 1〜2段落]
[アイキャッチ画像]

イベント概要          ← 定義リスト形式
総括                  ← 箇条書き 5〜10項目

─── 本文セクション（内容に応じて選択）───
自分のトークセッション（登壇した場合）
開場〜オープニング / キーノートとトーク / ランチ / クロージング〜懇親会
  または DAY1 / DAY2 ... の日別形式

写真で紹介（任意）
イベント感想          ← 末尾に必ず「次のイベント」への言及で締める
おまけ（任意）
```

## 各セクションの書き方

### リード文

- 何月・どこで開催されたかを1文で述べる
- その国・地域のPyConの特徴や位置づけを補足（初開催か、他国との比較）
- 全体的な雰囲気を1〜2文で先出しする
- 過去参加の同イベントがあれば比較を入れる

### イベント概要（定義リスト）

```rst
:イベント名: 正式名称
:公式ウェブサイト: URL
:日付: YYYY年MM月DD日(曜日)  ← 複数日はDAY1/2/3を記載
:場所: 会場名, 都市名, 国名
:参加費: 金額（国内イベントのみ）
```

直後に「参加時のメモ（blog元ネタ）がScrapboxにあります」＋URLを記載。

### 総括

箇条書き 5〜10 項目。各項目 2〜4 文。以下を網羅する:

- 雰囲気の総評（日本のPyConとの比較、質疑応答の活発さ）
- WiFiの有無
- コーヒー・飲食（提供タイミング、量、品切れ等）
- ランチ（現地料理の内容・辛さ・量）
- 懇親会・パーティー（お酒の有無、盛り上がり）
- 日本からの参加者・登壇者数（海外イベントの場合）
- スポンサーブースの様子
- 自分の登壇への反応（声をかけてもらえた話）

### 自分のトークセッション（登壇した場合）

```rst
- 日時場所
- Video: URL（"準備中" でも可）
- Slide: `"トークタイトル" <URL>`

- "トークタイトル"  ← 内容要約 2〜4文

- 質疑応答
  - Q&A 2〜4個
  - トーク後に声をかけられたエピソード
```

### 画像の書き方

```rst
.. figure:: ファイル名.*
   :width: 80%

   キャプション（文脈・情報・ユーモアを加える）
```

- ローカル画像はファイル名を `.*` で拡張子省略
- Scrapbox画像はフルURLを使用
- ツイート/X引用は「@username - 午後X:XX · 年月日 URL」形式でキャプションに入れる

### イベント感想（末尾）

- 3〜5段落、個人的な感情・達成感を素直に書く
- 感嘆文（「楽しかった！スピーカーやって良かった！」）を使ってよい
- **必ず次のイベントへの言及で締める**（「次は○○に行きたい」「プロポーザルを出した」「英語がんばろー」等）
- 過去記事は `:doc:` ディレクティブでリンクする

## 文体ルール

| 項目     | ルール                                                             |
| -------- | ------------------------------------------------------------------ |
| 敬体     | デス・マス体（〜でした、〜と思います）で統一                       |
| 感情表現 | 感嘆符（！）を惜しみなく使う                                       |
| ユーモア | 自虐・口語（「めちゃくちゃ」「ガンガン」「〜なー」）を適度に入れる |
| 食べ物   | 具体的な料理名・値段・辛さ・量を書く                               |
| 伝聞     | 「〜とのこと。」「〜だそうです。」を多用                           |
| 技術     | トーク紹介では質疑応答の具体例を入れて技術内容を要約する           |
| 継続性   | 過去記事への言及・「伏線回収」的フレーズを入れる                   |

## 参加メモ → blog記事のマッピング

| Scrapbox/メモの内容        | 対象セクション                       |
| -------------------------- | ------------------------------------ |
| イベント日時・会場・URL    | イベント概要                         |
| 全体の印象・雰囲気         | 総括                                 |
| キーノート・各トークのメモ | 日別/トーク別セクション              |
| 質疑応答メモ               | 「質疑応答」サブセクション           |
| 食事・コーヒー・おやつ     | 総括の箇条書き + 写真キャプション    |
| 懇親会メモ                 | クロージング〜懇親会セクション       |
| 自分のトーク後の反応       | 自分のトークセッションセクション     |
| 移動・宿泊メモ             | おまけセクション                     |
| 写真（URL）                | 各セクション内のfigureディレクティブ |
