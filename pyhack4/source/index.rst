Python Hack-a-thon 4 ハンズオン 中級コース
===========================================

担当: 清水川 http://www.freia.jp/taka/

このサイトの短縮URLは `http://縮.jp/一決 <http://縮.jp/一決>`_ です。


まえがき
---------

このハンズオンでは、以下の内容について体験していきます。

1. virtualenv: Python本体からの独立
2. setuptools: pluginを作ってみる
3. PyPI: pluginシステムを公開
4. buildout: Python本体からの独立２

環境
-----

* 出来ればUnix系。Windowsでも可だがハマリ多し。
* Pytohn2.6推奨。easy_installしまくってるとはまるかも。
